# Steem node RPC scanner

by @someguy123

Version 1.0

Python 3.7.0 or higher recommended

# Install

```
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
cp nodes.txt.example nodes.txt
# normal run
./app.py
# verbose output
./app.py -v
```
